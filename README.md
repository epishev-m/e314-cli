# e314-cli

CLI tool for working with e314 Unity modules.

## Quick installation (Windows)

```powershell
# Run the build and install script
.\build-tool.bat
```

## Usage

```bash
# Initialize project
e314 init

# List modules
e314 ls

# Clone all modules
e314 clone all

# Clone modules by indexes
e314 clone indexes

# Create new UPM module
e314 new upm

# Create new UPM project
e314 new upm-project
```
